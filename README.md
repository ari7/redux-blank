
# Redux Blank

An empty Redux project relying on [react-scripts][rs].

## Directory structure

```
.
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── main.css
└── src
    ├── components
    │   └── HelloButton.js
    ├── containers
    │   └── App.js
    ├── reducers
    │   └── index.js
    └── index.js
```


## Usage

Start with Yarn or NPM.

```
yarn start

# or 
npm start
```

[rs]: https://github.com/facebookincubator/create-react-app "a node.js script"
